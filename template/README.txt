SPICEファイルについて

基本的にSPICEのネットリストの書き方に習って書けば
問題ないです。

STL Circuit以下でSTLの回路を記述します。
STL化したい部分の名前に"_STL_num"を加えてください。
numは分割する数を表しています。
STLの場合
　W_STL_5   node1   0   node2   0    RLGCMODEL=Z50   N=1   L=300m
このように書くと300mm(30cm)の伝送路を5分割することを表します

C-STLの場合
　C_STL_1   node1   0   C=10p
このように書くと10pFのキャパシタを挿入することになります。
Cは分割できないので、numは1にしてください。
また、SPICEでは"C="がなくてもキャパシタンスを認識しますが、GAを実行する際に
必要なのでキャパシタンスは"C="をつけて書いてください。

C-STLでスコア計算にeye_sizeを用いる場合、寄生容量以外のCは初期値を0にしてください。

Versus Circuit以下には誤差面積を用いる場合の、基準となる理想信号を
伝送する回路を記述します。
スコア計算にeye_sizeを用いる場合はこの部分はあってもなくても問題ないです。

SPICEファイルを作る場合はdefault.spをコピーして必要に応じて編集すると
いいと思います。
